Implementation Details

This demo uses the LiquidImageReveal class with Three.js and custom GLSL shaders.

  • Vertex Shader: Pass-through with UV coordinate forwarding
  • Fragment Shader: Complex multi-effect pipeline
  • Uniforms: Dynamic parameters updated per frame
  • Mouse Tracking: Smooth interpolation for organic motion
  • Progress Animation: Eased transitions between states
  • Fallback Mode: CSS-based effects for browsers without WebGL
